# Sn Plugins Documentation

This is the shared documentation hub for every plugin in the Sn family whose releases are published in this repository, [Sn-Releases](https://github.com/ValentinTarnovsky/Sn-Releases).

Each plugin has its own section in the sidebar. Click a plugin to expand its full documentation: overview, installation, commands, permissions, configuration, and (when applicable) PlaceholderAPI placeholders and a FAQ.

{% hint style="info" %}
Looking for **SnLib** itself, the engine every Sn plugin is built on? It has its own separate documentation space. See the [SnLib repository](https://github.com/ValentinTarnovsky/SnLib) for the link.
{% endhint %}

New plugin sections are added automatically whenever a plugin ships a release through this repo, so this page always reflects what is currently available.
